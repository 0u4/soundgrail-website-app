Introduction to Slapform Docs

Slapform is a form processing service which allows you to handle forms on your websites & apps without any back-end code!

Getting started

Here's what you need to get started

  1. A website and access to the HTML code
  2. A Slapform account. Create one now.
  3. A basic understanding of HTML and JavaScript.

There are a variety of ways to create a form on Slapform.

You can create:

  • An HTML form by pointing the action attribute to our endpoint
  • A JavaScript / AJAX form by using our npm module or making a post request to our endpoint
